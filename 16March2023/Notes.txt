


HTML:

HyperText Mark up language is used for describing static web pages. 

Mark Up language: 

Mark up language contains a list of tags to represent data / ui. 

ex: HTML, XML

Tag: A tag is a keyword that is enclosed using angular brackets. 

ex: <html>, <head>, <body>....

HTML provides a set of tags to describe web pages.

How browsers execute / render web pages. 

When we give the web page to a browser, browser creates 
a tree (Data structure) called DOM Tree. 

DOM(Document Object Model):  It is a data structure tree that 
represent each tag of html page in the form of objects. 

Each tag is represented as an object.
