Environment Set up:

1. Install NodeJS 

https://nodejs.org/en 

2. open cmd and type node -v

3. create a folder in desktop / anywhere 

4. open that folder with VSCode 

5. npx create-react-app firstapp 

6. cd firstapp 

7. npm start
